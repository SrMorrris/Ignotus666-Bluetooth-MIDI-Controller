Se presupone que ya has quemado bootloader en el atmel 1284 (por lo menos ya has montado el chip en una placa o proto-board con el cristal de cuarzo,condensador,etc..)
Si no has quemado bootloader, luego deberás usar un programador para subir codigo, un programador como arduino UNO etc
Para poder subir codigo al 1284..

Descomprimir y meter la carpeta (puedes cambiar el nombre de la carpeta)
En c:/arduino/hardware

Despues deberás seleccionar en Arduino IDE en el apartado de Herramientas--PLACA
Selecciona "maniacbug Mighty 1284p 16Mhz using optiboot"
Solo basta con seleccionar el tipo de programador (para un FTDI, pondremos AVRISPMKII)
(arduinoasISP utilizando un arduino UNO como programador).
Y luego el puerto COM en el que este conectado el FTDI o el arduino UNO.
Y ya podremos subir codigo al atmel 1284.
